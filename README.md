# Shadowdark Adventures

This repository holds **original Shadowdark-compatible adventures and lairs**: Markdown-first authoring, with PDF export as a final build step (tooling TBD).

It is separate from **[ShadowdarkReference](https://github.com/pcons71/ShadowdarkReference)**, which stores personal reference material (rules page images, conversion notes).

## Contents

- `Shadowdark_Adventure_Template_Project_Checklist.md` — task list for the adventure template, toolchain, and legal/compliance prep.

## First-time publish to GitHub

Create an empty repository (for example `ShadowdarkAdventures`) under your account, then:

```text
git remote add origin https://github.com/YOUR_USER/ShadowdarkAdventures.git
git push -u origin main
```
