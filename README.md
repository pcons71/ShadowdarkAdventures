# Shadowdark Adventures

This repository holds **original Shadowdark-compatible adventures and lairs**: Markdown-first authoring, with PDF export as a final build step (tooling TBD).

It is separate from **[ShadowdarkReference](https://github.com/pcons71/ShadowdarkReference)**, which stores personal reference material (rules page images, conversion notes).

## Contents

- `Shadowdark_Adventure_Template_Project_Checklist.md` — task list for the adventure template, toolchain, and legal/compliance prep.

## Repository

Canonical remote: **[github.com/pcons71/ShadowdarkAdventures](https://github.com/pcons71/ShadowdarkAdventures)**.

```text
git clone https://github.com/pcons71/ShadowdarkAdventures.git
```

Local `origin` should track that URL; new clones get `main` by default.

## Related

- Intended to sit next to `ShadowdarkReference` under the same parent folder (e.g. `CascadeProjects`).
- **Multi-root workspace:** save `Shadowdark.code-workspace` in that **parent** folder (above both repos) and open it in Cursor/VS Code. Copy the JSON from the [ShadowdarkReference README](https://github.com/pcons71/ShadowdarkReference/blob/main/README.md) (*Multi-root workspace* section).
